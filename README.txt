===Chineses Ver.===
函式庫：	wavfile
來源：	https://www3.nd.edu/~dthain/courses/cse20211/fall2013/wavfile/

注意事項：
	不要亂改wavfile.c及wavfile.h的內容

使用方法：
1.)如果你知道makefile是什麼的話：
		想辦法執行makefile吧

2.)如果你不知道makefile是什麼 ：
		就別管那個makefile了
		去你最習慣的IDE(例如devC++、Visual Studio)，
		建立一個新的專案，並且新增三個新項目(兩個.c、一個.h)
		將sample_sound.c、wavfile.c的內容放到新增的兩個.c檔裡
		同樣把wavfile.h的內容放到新增的.h檔裡
		執行有sample_sound.c內容的.c檔
		完成

	#注意：因為我沒用過太多的IDE，所以步驟2.) 可 能 有誤。若有任何新的發現，歡迎回報～

===English Ver.===

Library：	wavfile
Source：	https://www3.nd.edu/~dthain/courses/cse20211/fall2013/wavfile/

Attention：
		Do NOT modify any words inside both “wavfile.c” and “wavfile.h”

Instructions：
1.)For those who know what “makefile” is ：
	Run the makefile directly. 

2.)If not：
	Ignore this makefile.
	Go and use the IDE with which you’re the most familiar
	Add new project, and add three new items (two for .c file, one for .h file) afterward
	Copy the contents of “sample_sound.c” and “wavfile.c” to those two .c files that you just added individually
	Similarly, copy content of “wavfile.h” to the newly added .h file
	Run the .c file that contains the content of sample_sound.c
	Done
	
	#Notice：Due to the fact that I’m not acquainted with too many IDE’s, there might be some mistakes during the procedure of step 2.). All feedback is welcome.
